# 🌐 웹 애플리케이션 설정 가이드

Streamlit을 사용한 차량 감지 웹 애플리케이션 실행 방법입니다.

## 🚀 빠른 시작

### 1. 의존성 설치
```bash
# Streamlit 추가 설치
pip install streamlit pandas

# 또는 업데이트된 requirements.txt 사용
pip install -r requirements.txt
```

### 2. 웹 애플리케이션 실행
```bash
# 기본 실행
streamlit run app.py

# 특정 포트로 실행
streamlit run app.py --server.port 8501

# 외부 접속 허용
streamlit run app.py --server.address 0.0.0.0
```

### 3. 브라우저에서 접속
```
http://localhost:8501
```

## 🎛️ 웹 인터페이스 기능

### **📸 이미지 분석 탭**
- **이미지 업로드**: 드래그 앤 드롭 또는 파일 선택
- **실시간 분석**: 업로드와 동시에 자동 처리
- **결과 시각화**: 다양한 형태의 결과 표시
- **상세 정보**: 감지된 각 차량의 정보 테이블

- **결과 다운로드**: 텍스트, 이미지, JSON 형태로 다운로드

### **📚 사용 가이드 탭**
- **빠른 시작 방법**
- **설정별 최적화 팁**
- **지원하는 차량 종류**
- **문제 해결 방법**

- **성능 모니터링**: 메모리 사용량 및 시스템 상태

## ⚙️ 사이드바 설정

### **🤖 모델 설정**
- **YOLO 모델**: Nano(빠름) ~ Large(정확) 선택
- **SAM 활성화**: 고급 세그멘테이션 on/off
- **SAM 모델**: ViT-H(최고품질) ~ ViT-B(빠름) 선택

### **🔍 감지 설정**
- **신뢰도 임계값**: 0.1 ~ 0.9 조정
- **처리 디바이스**: 자동/GPU/CPU 선택

### **ℹ️ 실시간 정보**
- **GPU 상태**: 사용 가능 여부 표시
- **메모리 사용량**: 실시간 모니터링
- **유용한 링크**: 공식 문서 바로가기

## 🎯 사용 팁

1. **첫 실행 시** 모델 다운로드로 시간이 걸릴 수 있습니다
2. **GPU 사용 시** 첫 번째 추론이 느릴 수 있습니다
3. **대용량 이미지**는 자동으로 리사이즈됩니다
4. **배치 처리** 시 메모리 사용량을 모니터링하세요
